Opdracht 1

  • "Block" zet alle "area's" onder elkaar.
  • "Inline" Maakt alle blokjes kleiner en zet ze allemaal naast elkaar.
  • "Inline-Block" Maakt ze de "area's" passend aan de pagina, de "area's" die er niet meer op kunnen worden onder de 1e lijn gezet.
  • Bij "Block" en "Inline-Block" worden de height en width toegepast.

Opdracht 3

  • In de stylesheet staat er voor elk "div" element dat de tekst gecentreerd moet staan.
  • In het "div" element is er geen regel die een line-height definieert dus heeft de center aligning geen effect.
  • Het "vertical-allign" element zorgt ervoor dat de middens van de "div" elementen worden ge-alligned.
  • Zorg ervoor dat er een "line-height" element is waardoor je "text-allign" kan gebruiken.
  • Je stelt eerst een display in als table cell waardoor het zal werken als een tabel in een cell.
  • Tall krijgt voorrang omdat hij onderaan in de css code van "area" staat.
